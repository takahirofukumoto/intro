Gitの練習をしていきます
基本的なワークフローを学びます
自分の力でやってみる
cd ~/Desktop
　・デスクトップに移動
cd フォルダ名
　・フォルダへ移動
ls
　・現在のフォルダの中身を確認
ls -a
　・現在のフォルダの中身を隠しファイル含めすべて確認
git add 
　・ステージングエリアに追加
git add .
　・ステージングエリアにフォルダ内すべての変更フォルダを追加
git commit
　・ステージングエリアからローカルリポジトリへコミット
　・VIM内では1行目に変更の概要2行目空3行目に変更理由が良い
　・「Esc」→「:qw」でコミット
git commit -v
　・コミットする際に、DIFFを確認できる
git push -u origin master
　・リモートリポジトリへプッシュ
git status
　・現在のフォルダ内でのadd状況など確認できる
git log
　・コミット内容を確認できる
