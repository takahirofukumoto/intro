Gitの練習をしていきます
基本的なワークフローを学びます
自分の力でやってみる
cd ~/Desktop
　・デスクトップに移動
cd フォルダ名
　・フォルダへ移動
ls
　・現在のフォルダの中身を確認
ls -a
　・現在のフォルダの中身を隠しファイル含めすべて確認
git add ファイル名
　・ステージングエリアに追加
git add .
　・ステージングエリアにフォルダ内すべての変更フォルダを追加
git commit
　・ステージングエリアからローカルリポジトリへコミット
　・VIM内では1行目に変更の概要2行目空3行目に変更理由が良い
　・「Esc」→「:wq」でコミット
git commit -v
　・コミットする際に、DIFFを確認できる
git push -u origin master
　・リモートリポジトリへプッシュ
git status
　・現在のフォルダ内でのadd状況など確認できる
git log
　・コミット内容を確認できる
　・「j」で上、「ｋ」で下へ移動可能
　・ログの確認を終わるときは「q」で可能
git log --oneline
　・各コミット内容を1行で確認できる
git log -n 3
　・直近のコミットを何個表示するか指定してログを確認できる
　・git log -n 3 --onelineも可能
git rm　
　・対象を削除としてステージングエリアに追加
